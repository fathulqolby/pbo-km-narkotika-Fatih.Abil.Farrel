# pbo-km-narkotika-Fatih.Abil.Farrel
Proyek ini adalah sebuah aplikasi berbasis Java (JDK 11+) yang dirancang untuk merancang dan mengimplementasikan Knowledge Management System (KMS) pada Putusan Pengadilan Narkotika. Aplikasi ini menerapkan konsep Pemrograman Berorientasi Objek (OOP) secara mendalam, termasuk enkapsulasi, polimorfisme, dan pewarisan.
